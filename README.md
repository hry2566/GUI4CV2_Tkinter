# GUI4CV2_Tkinter  
GUI for OpenCV-Python with Tkinter  
・OpenCV-Pyhtonの簡易設定画面を持ったツールです。  
・画面はpygubu-designer(Tkinter)で作成されています。  
・画像処理は素人なので間違いがあるかもしれません。  
  
必要なライブラリ  
・OpenCV-Python  
・numpy  
・tkinter
